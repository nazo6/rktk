---
title: Dongle
---

Dongle is usually connected to a computer via USB. It receives data from the wireless keyboard and sends it to the host. It can also be used to send data from the host to the keyboard.

## Driver list

### nRF

:::drivers_table

| name       | crate            | path       | description                                                                                                                                     |
| ---------- | ---------------- | ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| 2.4GHz ESB | rktk-drivers-nrf | esb/dongle | Received 2.4GHz ESB Reporter data from the keyboard and sends it to the host. It is intended to use with keyboard that has ESB reporter driver. |

:::
