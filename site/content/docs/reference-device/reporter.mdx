---
title: Reporter
---

Reporter drivers are used to send data from the device to the host. This can be done over USB, BLE, or other protocols.

## Driver list

### Common

:::drivers_table

| name          | crate               | path | description                                                                                                       |
| ------------- | ------------------- | ---- | ----------------------------------------------------------------------------------------------------------------- |
| USB (embassy) | rktk-drivers-common | usb  | USB reporter Driver based on embassy-usb. Can be used with any platform that embassy-usb's driver is implemented. |

:::

### nRF

:::drivers_table

| name             | crate            | path         | description                                                                         |
| ---------------- | ---------------- | ------------ | ----------------------------------------------------------------------------------- |
| BLE (softdevice) | rktk-drivers-nrf | softdevice   | BLE reporter driver based on the nRF softdevice. Can be used with any nRF platform. |
| 2.4GHz ESB       | rktk-drivers-nrf | esb/reporter | Reports to other nRF device connected to PC using proprietary ESB protocol.         |

:::
